# Matrix Theme

**Author:** BillyJBryant  
**Version:** 1.0  
**Firmware Version:** 1.04

## Description

A Matrix-inspired theme for the WiFi Pineapple Pager featuring the iconic green-on-black aesthetic. 

## Installation

1. Copy the `matrix` theme directory to `/root/themes` on your Pager
1. Select the theme from the Pager settings
